Adapter is a special object to assess and manipulate the uiScroll. The API it provides is being discussed on this page. The only way to receive the Adapter is to instantiate the Datasource via operator new: 

{{datasourceSample}}

The constructor argument is an object of IDatasource type which must include get method property and may include settings object property. The uiScroll augments the result Datasource object with the Adapter object property during instantiating. So after the Datasource object had been instantiated, we may access this.datasource.adapter property object with its API methods and properties.
